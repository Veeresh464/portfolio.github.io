Sorting Algorithms

Timsort

Timsort is sorting algorithm used in real-world programming languages like Python and Java. It works by looking for small already sorted parts (called runs) in the data and then combining them smartly.

If it finds a part that's already sorted, it keeps it that way and just builds around it. It mixes the best ideas from Insertion Sort (good for small data) and Merge Sort (good for big data).

Worst-case Time: O(n log n) | Space: O(n)

Counting Sort

Counting Sort is very fast but only works when you're sorting numbers that are not too big. It doesnâ€™t compare elements like other sorting methods. Instead, it counts how many times each number appears, then uses that information to figure out where each number goes.

Explain??

Counting Sort is a fast algorithm used when the values you're sorting are integers and fall within a known, limited range.

Steps :

  • Find the biggest number in your list.
  • Create a new array to count how often each number appears.
  • Go through the original list and update your count array.
  • Rebuild the sorted list using that count information.
Example: For the array [1, 2, 2], the count array becomes [0, 1, 2].

JavaScript Code


// Counting Sort
function countingSort(arr) {
  const max = Math.max(...arr);
  const count = new Array(max + 1).fill(0);

  // Count each element in the array.
  for (let i = 0; i < arr.length; i++) {
    count[arr[i]]++;
  }

  let index = 0;

  // Reconstruct the array using the count array.
  for (let i = 0; i <= max; i++) {
    while (count[i] > 0) {
      arr[index] = i;
      count[i]--;
      index++;
    }
  }

  return arr;
}

// Try it:
countingSort([8, 5, 2, 9, 5, 6, 3]);

It's like counting how many people got each score in a test and then arranging them in order.

Merge Sort

Merge Sort breaks your list into two halves again and again until each part has just one item. Then it starts merging them back in order.
