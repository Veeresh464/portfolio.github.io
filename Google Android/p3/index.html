Functionalities of Android OS

Memory management

Android uses a paging system with OOM killer to manage memory efficiently across apps.

Key Algorithms in Android Memory Management

  • Paging & Swapping
  • Low Memory Killer (LMK) / OOM Reaper
  • ZRAM (compressed RAM)
  • Kernel Samepage Merging (KSM)
  • ASHMEM (Android Shared Memory)

Process scheduling

Utilizes the Linux Completely Fair Scheduler (CFS) with Android-specific optimizations.

Key Features in Android Process Scheduling

  • Completely Fair Scheduler (CFS)
  • Real-time Scheduling with SCHED_FIFO
  • Binder IPC Thread Prioritization
  • CPUSet & Task Groups for process isolation
